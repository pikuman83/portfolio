Overview

This PWA not only streamlines financial reporting, it also combines multiple sources of data to provide a complete and accurate financial picture providing a high-level visualization of KPIs, using real-time data within any range of dates, directly from the dashboard. It puts the power of sophisticated data analytics for advanced forecasting, budgeting and planning directly into the hands of executives or financial analysts.

Key Challenges

Performing calculations on large datasets. Pitch and Performance tracking data visualisation, according to different sets of parameters. Presenting users with the opportunity to manage, analyse and optimise data in runtime and real-time. Some other technical challenges were securing the API, State Control using singleton Services, asynchronous task processing, SAP crystal reports incorporation using FOC tools & Ngx-charts Incorporation, Default date ranges, scalability and Hosting.

Development & Tools

  • The frontend is developed using Angular framework, ASP.Net is used to connect to the MSSQL database, to create resources and to read and export reports made in SAP Crystal Reports
  • In terms of languages, the code marked up by HTML has been decorated through CSS and Bootstrap, TypeScript & JavaScript has brought the logic to the frontend under Angular Framework, C# has taken care of the backend and SQL Queries to analise data
  • The app is hosted on the company's personal server through IIS
  • Local storage, Session storage and Singleton Services are used to store and handle the data on the client side
  • A Splash screen showing log is used with "Resolver" to load the data and dashboard on the app start
  • Ngx-charts to enhance the view of the landing page
  • Angular material has brought the elegance
  • Frontend routes has been secured by guard and http interceptor, the user is validated through JWT. Also a 20 minutes inactivity logout at the client side and a 2 hours token expiration has been added
  • For the web, manifest.webmanifest has personalized the app installation, also a manual installation prompt and instructions has been added

Timeline

Project started at the beginning of January 2021 and took less than 30 days to be completed. Some new reports were ordered by the client and were incorporated by the time.
